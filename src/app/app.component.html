Factorial

The factorial of a number is the product of a number and all the numbers below it.
Enter a positive number: 

                            $("#factbtn").click(function () {
                                var fnum = +$("#nfact").val();
                                var fact = 1;

                                for (let i = 1; i <= fnum; i++) {
                                    fact *= i;
                                }

                                $("#res").text(fact);
                            });

Fizz Buzz

Finds the multiples of the numbers that are entered and prints it out as fizz, buzz, or fizz-buzz if both numbers share the same multiple.
Enter two numbers between 1 to 100. 

                        $("#fbuzzbtn").click(function () {
                            let fbArray = new Array();

                            var fb1 = +$("#fbuzz1").val();
                            var fb2 = +$("#fbuzz2").val();
                            var fizz = 0, buzz = 0;

                            for (let i = 1; i <= 100; i++) {
                                fizz = i % fb1;
                                buzz = i % fb2;
                                if (fizz == 0 && buzz == 0) {
                                    fbArray.push("Fizz-Buzz");
                                }
                                else if (fizz == 0) {
                                    fbArray.push("Fizz");
                                }
                                else if (buzz == 0) {
                                    fbArray.push("Buzz");
                                }
                                else {
                                    fbArray.push(` ${i}`);
                                }

                                fbArray.join(',');
                            }
                            $("#fbresult").text(fbArray);

                        });

Palindrome

A Palindrome is a word that is spelled the same forwards as it is backwards.
Enter a word: 

                    $("#palbtn").click(function () {
                        var pal = $("#palinput").val();
                        var result;

                        let wordArray = pal.split('');          // converts to array
                        let revArray = wordArray.reverse('');   // reverses word
                        result = revArray.join('');             // stores reversed word into string

                        if (pal === result) {                   // compare reversed string to original
                            $("#resultpal").text(pal + " is a palindrome!");
                        } else {
                            $("#resultpal").text(pal + " is not a palindrome.");
                        }

                    });

Math Calculations

Calculates the largest number, smallest number, sum, product, and mean of 5 numbers.
Enter 5 numbers: 


                            $("#Sumbtn").click(function () {
                                var num1 = +$("#num1").val();
                                var num2 = +$("#num2").val();
                                var num3 = +$("#num3").val();
                                var num4 = +$("#num4").val();
                                var num5 = +$("#num5").val();
                                var sum, product, output, mean, largest, smallest;

                                // smallest number
                                smallest = Math.min(num1, num2, num3, num4, num5);
                                //largest number
                                largest = Math.max(num1, num2, num3, num4, num5);
                                //mean
                                mean = (num1 + num2 + num3 + num4 + num5) / 5;
                                //sum
                                sum = num1 + num2 + num3 + num4 + num5;
                                //product
                                product = num1 * num2 * num3 * num4 * num5;

                                output = "small: " + smallest + ", large: " + largest + ", sum: " + sum + ", product: " + product + ", mean: " + mean;

                                $("#result").text(output);
                            });
